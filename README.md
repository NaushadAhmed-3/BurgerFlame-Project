##  Usage Instructions
1. Download the repository or clone it using Git.
2. Unzip the folder into any of your Directory.
3. Open the Folder into `Vs Code`.
4. Open the `index.html` file in any modern web browser (Chrome, Edge, Firefox).
5. Navigate using the menu bar to view the  Home, Menu, Cart, About Us, SignUp and Contact pages.
